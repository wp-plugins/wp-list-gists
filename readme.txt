Output Gists from Github with ease!

== Description ==

WP List Gists takes any given GitHub username and returns the users public Gists in a custom field on any of your selected post types.

== Installation ==

1. Upload `wp-list-gists` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Generate a Personal API Access GitHub token (https://help.github.com/articles/creating-an-access-token-for-command-line-use)
4. Go to Settings > WP List Gists. Enter your GitHub username, Token and select the post types you'd like to apply the plugin to.
5. Head to a post and you'll see the WP List Gists panel listing all your Gists. You can temporarily receive Gists from another user by supplying a new username in the username field.
6. Click a Gist to place a shortcode in TinyMCE.
7. Publish and profit!

== Frequently Asked Questions ==
