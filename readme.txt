Lists all Gists in your Posts and Pages from a given username and allows you to add them to your content in a shortcode form.

== Description ==

List any Gist from GitHub via a shortcode. Gives a list of shortcodes from any given username while editing a post. Great for anyone who needs to display code. You can temporarily get a list of shortcodes from another user, just supply a username.

The shortcode is also available for any other Github Gists, just supply a ID and User.

Uses the GitHub Gists loader to display the Gist.

Please note: You will need to generate a Personal Access Token in GitHub otherwise you will be restricted by the API request limit.

== Installation ==

1. Upload `wp-list-gists` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Settings > WP List Gists and set up your GitHub personal access token.
4. Select the post types you would like the WP List Gists selector to show in.
5. Profit!
